# EV Connect - Quick Start Guide

## 🚀 Getting Started

### Opening the Application

Simply open `index.html` in your web browser:

**Option 1: Double-click the file**
- Navigate to `c:\VS CODE\EV Connect\`
- Double-click `index.html`

**Option 2: Use a local server (recommended for best experience)**
```bash
# Using Python 3
cd "c:\VS CODE\EV Connect"
python -m http.server 8000

# Then open: http://localhost:8000
```

**Option 3: Open in browser directly**
- Open your browser
- Press `Ctrl+O` (or `Cmd+O` on Mac)
- Navigate to `c:\VS CODE\EV Connect\index.html`
- Click Open

### First Time Setup

1. **Welcome Screen** appears first
   - Click "Get Started" to begin
   - Or click "Explore Map" to skip authentication

2. **Create Account**
   - Enter your details in the Sign Up form
   - Choose your user type:
     - **EV User**: For finding and booking charging stations
     - **Charger Owner**: For managing your charging stations
   - Click "Create Account"

3. **Start Exploring**
   - EV Users → Map screen with nearby chargers
   - Charger Owners → Dashboard with statistics

## 📱 Quick Navigation Guide

### Bottom Navigation Bar (EV Users)
- 🗺️ **Map** - Find charging stations
- 📅 **Bookings** - View your reservations
- 👤 **Profile** - Manage account and vehicles

### Key Buttons
- 🌙 **Theme Toggle** - Switch between light and dark mode (top right on map)
- 🔍 **Search** - Find locations (top of map screen)
- ⚙️ **Filters** - Refine charger search (next to search bar)
- ❤️ **Favorite** - Save chargers (on details screen)

## 🎯 Common Tasks

### Booking a Charging Session

1. Open the **Map** screen
2. Click on any **green pin** marker
3. Review the charger details in the preview card
4. Click **"View Details"**
5. Scroll to see amenities and time slots
6. Click **"Book Now"**
7. Select your preferred **date and time**
8. Choose your **vehicle**
9. Select **payment method**
10. Click **"Confirm Booking"**

### Managing Your Profile

1. Click **Profile** in the bottom navigation
2. **Edit Profile** - Update personal information
3. **Add Money** - Top up your wallet
4. **Add Vehicle** - Register new vehicles
5. Check **Booking History** - Review past sessions

### For Charger Owners

1. View **Dashboard** statistics
2. Click **"Add Charger"** to register a new station
3. **Manage** individual chargers
4. **Accept/Decline** booking requests
5. Monitor **earnings** and **ratings**

## 🎨 Customization

### Theme
Toggle between light and dark mode using the sun/moon icon on the map screen. Your preference is saved automatically.

### Filters
Customize your charger search:
- **Distance**: 5km, 10km, 25km, or any distance
- **Price**: Adjust the slider
- **Availability**: Available now, Fast charging, 24/7 access
- **Power Type**: AC, DC, or Both
